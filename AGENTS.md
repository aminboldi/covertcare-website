# AGENTS.md

## Package manager

**pnpm only.** npm and yarn are blocked by the `preinstall` script. Always use `pnpm`.

## Commands

| Task | Command |
|------|---------|
| Full build (typecheck + build all) | `pnpm run build` |
| Typecheck only (libs + artifacts + scripts) | `pnpm run typecheck` |
| Typecheck lib packages only (faster) | `pnpm run typecheck:libs` |
| Build without typecheck | `pnpm -r --if-present run build` |
| Regenerate API code from OpenAPI spec | `pnpm --filter @workspace/api-spec run codegen` |
| Push DB schema changes | `pnpm --filter @workspace/db run push` |
| Run tests (main site only) | `pnpm --filter @workspace/covertcare-site run test` |

**No lint command exists.** Prettier is installed but not wired into any script.

Build order matters: `typecheck` runs before `build` in the root `build` script. Lib packages must typecheck first (they're composite project references).

## Architecture

```
lib/api-spec/          OpenAPI spec + Orval codegen config
lib/api-zod/           Generated Zod schemas (composite ts ref)
lib/api-client-react/  Generated React Query hooks (composite ts ref)
lib/db/                Drizzle ORM client + schema (composite ts ref)
---|---
artifacts/api-server/  Express 5 API (esbuild bundle)
artifacts/covertcare-site/  Main website (Vite 7 SPA)
artifacts/pitch-deck/  Pitch deck (Vite 7 SPA)
artifacts/mockup-sandbox/  UI sandbox (Vite 7)
```

**Dependency chain:** `api-spec` â†’ (codegen) â†’ `api-zod` + `api-client-react` â†’ `api-server` (uses `api-zod` + `db`) / `covertcare-site` (uses `api-client-react`).

`pitch-deck` and `mockup-sandbox` have no workspace dependencies.

## Generated code

**Never edit these directories by hand:**

- `lib/api-zod/src/generated/`
- `lib/api-client-react/src/generated/`

Both are generated by `pnpm --filter @workspace/api-spec run codegen` from `lib/api-spec/openapi.yaml`. After changing the spec, run codegen then typecheck.
